openvpn

Installs and configures OpenVPN

Requirements
------------

This role requires Ansible 1.4 or higher.

Role Variables
--------------

None

Dependencies
------------

None

Example Playbook
----------------

Install OpenVPN
```
- hosts: servers
  roles:
    - { role: kbrebanov.openvpn }
```
